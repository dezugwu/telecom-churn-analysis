# Telco Customer Churn Analysis

This repository contains two different approaches to solving the same Telco Customer Churn problem:

## 📁 Projects

- **[Power BI Version](./PowerBI_Version)**  
  Interactive 6-page executive dashboard with rule-based predictive scoring, risk intelligence, financial impact analysis, and strategic recommendations.

- **[Python ML Version](./Python_ML_Version)**  
  Machine Learning approach using Python (scikit-learn, pandas, matplotlib, etc.) for churn prediction and modeling.

## Technologies Used

- **Power BI Version**: Power BI, DAX, Key Influencers
- **Python ML Version**: Python, scikit-learn, Jupyter Notebooks

# Telco Customer Churn Analysis

This repository contains **two different approaches** to solving the same Telco Customer Churn business problem using the same dataset.

## 📁 Projects

### Power BI Version
**[→ PowerBI_Version](./PowerBI_Version)**

Interactive 6-page executive dashboard built in Power BI featuring:
- Executive Overview with Revenue at Risk tracking
- Customer Behaviour Analysis
- Rule-based Predictive Model Results (63.46% Recall)
- Risk Intelligence with Key Influencers
- Financial Impact Analysis
- Strategic Recommendations

**Live Dashboard**: [View Power BI Report](https://bit.ly/3PsTLY1)

### Python ML Version
**[→ Python_ML_Version](./Python_ML_Version)**

Machine Learning approach using Python (scikit-learn, pandas, Jupyter Notebooks) for churn prediction and modeling.

## Technologies Used

- **Power BI Version**: Power BI, DAX, Key Influencers Visual
- **Python ML Version**: Python, scikit-learn, pandas, matplotlib, Jupyter

## About This Project

This project demonstrates versatility in solving the same business problem using different tools — from interactive executive dashboards in Power BI to machine learning modeling in Python.

Both implementations focus on identifying churn drivers, quantifying revenue risk, and providing actionable retention strategies.
